# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| main    | ✅        |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report vulnerabilities by email to: **davide.silvestri15@yahoo.it**

Include in your report:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

**Response timeline**: acknowledgement within 72 hours, patch timeline communicated within 7 days.

## Scope

In scope: authentication bypass, RLS policy gaps, API key exposure, injection attacks.  
Out of scope: issues in third-party services (Supabase, Replicate, OpenAI, etc.).
